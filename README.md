Green Shoes
=========

This is *NOT* [Shoes](http://shoes.heroku.com/), but looks alike. So, named **Green Shoes**. This is written in Ruby. ;-)

Fantastic logo icon was created by [Zachary Scott](http://github.com/zacharyscott). Thanks!
